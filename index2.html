Child Selector (>) :

The child selector (>) selects all elements that are the children of a specified element (present inside)

I am a child selector (>)

I select all the immediate children of a specific element

This line belongs to another specific element

Which means that in our case

I select all the p tag elements which are the children of the div tag


Adjacent Sibling Selector (+) :

The Adjacent sibling selector (+) selects an element that is directly after another specific element.

I am a Adjacent sibling selector (+)

I select the immediate element after the specific element

There is also one condition

that is the sibling elements must have the same parent element

which means that in our case

I select all the p tag elements which are directly next to the div tag

I am one of the directly next element of the specific tag

General Sibling Selector (~) :

The general sibling selector (~) selects all elements that are next siblings of a specified element.

I am a General sibling selector (~))

I select all the next sibling elements of a specific element

I am also one of the next sibling elemnt

that is the sibling elements must have the same parent element
